Backend server structure

1. Build a websocket to enable streaming of data, UI input > to transcription, transcribed text > UI
2. Chunk the streaming data and queue input
3. Use whisper to transcribe the chunks and and send the results back to the UI
4. use or create a framework that allows us to use agents to analyze the transcribed results
